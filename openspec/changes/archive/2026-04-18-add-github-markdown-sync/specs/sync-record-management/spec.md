## ADDED Requirements

### Requirement: 同步记录管理
系统 SHALL 记录每次同步操作的详细信息，包括同步时间、仓库ID、同步状态、处理文件数、成功数、失败数等。

#### Scenario: 查看同步历史列表
- **WHEN** 用户访问同步记录页面
- **THEN** 系统返回分页的同步历史记录，按时间倒序排列

#### Scenario: 查看同步详情
- **WHEN** 用户点击某条同步记录查看详情
- **THEN** 系统返回该次同步的详细信息，包括每个文件的处理状态、错误信息

#### Scenario: 删除同步记录
- **WHEN** 用户删除某条同步记录
- **THEN** 系统删除同步记录，关联的文件映射关系保留，已同步的文章不受影响

### Requirement: 同步文章管理
系统 SHALL 支持查看和管理所有通过同步导入的文章。

#### Scenario: 查看同步文章列表
- **WHEN** 用户访问同步文章管理页面
- **THEN** 系统返回所有同步导入的文章，支持按仓库、分类、标签筛选

#### Scenario: 删除同步文章
- **WHEN** 用户删除某篇同步文章
- **THEN** 系统删除文章记录，同时删除对应的同步映射关系，同步记录保留

#### Scenario: 批量删除同步文章
- **WHEN** 用户选择多篇同步文章进行批量删除
- **THEN** 系统批量删除选中的文章和对应的映射关系
