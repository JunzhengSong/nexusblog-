## Context

现有首页 `Home.vue` 结构简单：Header + 文章列表。ArticleCard 仅显示标题、标签、日期。需重构为极简主义风格首页。

技术栈：Vue 3 + Element Plus + 原生 CSS。

## Goals / Non-Goals

**Goals:**
- 极简主义风格，中心化内容布局
- 文章卡片显示标题 + 120 字符摘要 + 日期 + 标签
- 分类 Tab 筛选功能
- 响应式布局（桌面端优先）

**Non-Goals:**
- 不做移动端专项优化
- 不添加搜索、RSS、分页等复杂功能
- 不修改后端 API

## Decisions

### 1. 移除 ArticleCard 组件，内联卡片设计

**选择**：将卡片逻辑直接写在 Home.vue 中
**理由**：当前 ArticleCard 功能简单，内联可减少组件层级，提升可维护性

### 2. 分类筛选使用前端过滤

**选择**：获取所有文章和分类，前端根据选中分类过滤
**理由**：文章数量可控（前 76 篇），无需为筛选增加后端 API

### 3. 摘要截取纯文本处理

**选择**：直接截取 content 前 120 字符，去除 Markdown 语法
**理由**：简单直接，避免引入 Markdown 解析依赖

### 4. 布局容器最大宽度 900px

**选择**：内容区域最大 900px 居中
**理由**：极简风格不需要太宽，900px 适合阅读

## Risks / Trade-offs

| 风险 |  Mitigation |
|------|-------------|
| 文章数量增长后前端过滤性能下降 | 届时可改为后端分页/筛选 API |
| 120 字符截断可能切断单词 | 可接受，摘要本身即是预览 |

- 标签显示数量是否需要限制？（当前方案最多显示 3 个）
